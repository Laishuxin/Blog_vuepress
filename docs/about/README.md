该博客是基于vuepress + `vuepress-theme-hope`实现的。主要用于记录学习前端的点点滴滴，本博客不会涉及除其他后端编程语言。

写博客的初衷就是为了分享（事实上，这也是我学编程的初衷，分享与协助），在学习前端的过程中，受到很多博主的启发和帮助，在看到阮一峰老师写的文章时，发现他是真的用心去写每一篇博文，深受其影响，也想认真地写好每一篇博文，毕竟种一棵树最好的时间是十年前，其次就是现在。

本博客将主打系列文章。这是因为我在阅读[使用Typescript重构axios](https://www.cnblogs.com/wangjiachen666/p/11234163.html)过程中深受启发，一个好的博客，不应该是一系列零散的文章，而应该是用一个系列的文章逐步深入，为读者提供更好的学习体验。
所以，本博客的侧重点会倾向于系列文章。这些系列文章主要是我在阅读过一些好的书籍后的所思所想。同时，在每一个系列文章中，我也会给出相应的引用，方便读者阅读。

虽然目前的水平还有所欠缺，但是我会尽可能地把文章写好。
