## 常见请求方法
+ GET
+ POST
+ DELETE
+ PUT
+ HEAD
+ OPTIONS

## GET 与 POST 的区别

GET: 
+ GET请求可以被缓存
+ GET请求可以保留在浏览器历史记录中
+ GET请求可以被收藏为书签
+ GET请求**不应该**在处理敏感数据是使用
+ GET请求中URL有长度的限制
+ GET请求最好用于取回数据

POST： 
+ POST请求不会被缓存
+ POST请求不会保留在浏览器历史记录中
+ POST请求不能被收藏为书签
+ POST请求数据的长度没有要求
