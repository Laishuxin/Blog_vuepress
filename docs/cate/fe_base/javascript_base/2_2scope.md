## 词法作用域与动态作用域
### 作用域
> 作用域：定义变量的区域。作用域规定了如何查找变量。即，对变量的访问权限。

### 静态作用域与动态作用域
> 静态作用域：函数的作用域在定义时就决定了。（例如，js）
> 动态作用域：函数的作用域在函数执行的时候才决定。（例如：bash）
